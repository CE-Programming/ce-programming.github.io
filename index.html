C Libraries

The C Libraries provide methods for performing quick and painless graphics support, file IO operations, and keypad scanning assistance on the TI-84 Plus CE / TI-83 Premium CE calculators. An API comes integrated with the C toolchain for ease of use.

C Toolchain

The C Toolchain is a series of programs and tools used to build and create new programs using the C language for the TI-84 Plus CE / TI-83 Premium CE calculators. It comes with a heavy emphasis on documentation, with guides to help in all steps of program development.

CEmu

CEmu is a third-party TI-84 Plus CE / TI-83 Premium CE calculator emulator, focused on developer features. CEmu works natively on Windows, macOS, and Linux.
